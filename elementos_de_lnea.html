Elementos de Línea

  • Los elementos en línea, por otro lado, se utilizan para dar estilo o formato al contenido dentro de los elementos de bloque, como enlaces o imágenes.
  • No comienzan en una nueva línea y solo ocupan el espacio necesario para mostrar su contenido.
  • Esto permite que varios elementos en línea aparezcan en la misma línea, separados por un espacio.

Ejemplos de elementos en línea:

  • <a> → enlaces
  • <img> → imágenes
  • <span> → aplicar estilo o agrupar texto dentro de una línea
