Introducción a las Páginas WEB

1. ¿Qué es una Página WEB?

  • Una página web es un documento que puede contener texto, sonido, vídeo, programas, enlaces, imágenes, hipervínculos y muchas otras cosas, adaptada para la llamada World Wide Web (WWW) y a la que se puede acceder mediante un navegador web.
  • Esta información se encuentra generalmente en formato HTML y puede proporcionar acceso a otras páginas web mediante enlaces de hipertexto. 
  • Frecuentemente también incluyen otros recursos como pueden ser hojas de estilo en cascada (.css), scripts (.js), imágenes digitales(.jpg), etc.
  • Las páginas web pueden estar almacenadas en un ordenador local o en un servidor web remoto. 

2. ¿Qué significan las siglas HTML?

1. Significado: HTML son las siglas de HyperText Markup Language, en español Lenguaje de Marcado de Hipertexto.
2. No es un lenguaje de programación, sino un lenguaje de marcas que sirve para estructurar la información en una página web.
3. Para qué sirve: Indica al navegador (Chrome, Firefox, Edge…) qué mostrar y cómo organizarlo.
4. Por ejemplo: títulos, párrafos, imágenes, enlaces, tablas, listas…

3. ¿En qué consiste el protocolo HTTP?

  • Es un conjunto de normas y reglas que todos los navegadores entienden. Gracias a estas reglas, cualquier página HTML se puede ver en cualquier ordenador o móvil, sin importar el sistema operativo. Es un protocolo de comunicación que se usa para transmitir contenido entre el servidor y el navegador.
  • Idea clave:
    • HTML = qué contiene la web (estructura, textos, imágenes).
    • HTTP = cómo viaja la información de la web del servidor al navegador.
  • Ejemplo
    • Escribimos en Chrome: https://www.facebook.es.
    • El navegador se comunica con el servidor usando el protocolo HTTP/HTTPS.
    • El servidor envía un archivo escrito en HTML.
    • El navegador lee ese HTML y muestra la página web.

4. ¿Qué es la dirección URL y de qué partes se compone?

  • Para acceder a una página web almacenada en la web, necesitamos conocer su URL (Uniform Resource Locator) e introducirla en la barra de direcciones de un navegador web.
  • Cada página web tiene su propia URL. Las URL están compuestas de varias partes que permiten al navegador localizarlas dentro de un servidor.
    • Protocolo: Reglas que debe seguir el navegador para encontrar la página web.
    • Subdominio: Para el protocolo http/https suele ser www, aunque existen otros.
    • Dominio: Es el nombre donde se almacena el sitio web.
    • Subdirectorio: Es una carpeta alojada dentro del dominio indicado.
    • Archivo: Es el archivo o página que nos va a mostrar el navegador web. Está dentro del subdirectorio indicadO

5. ¿Qué son y para qué utilizan las Cookies los navegadores WEB?

  • Una cookie es un pequeño archivo de texto que una página WEB guarda en tu ordenador o móvil cuando la visitas.
  • ¿Para qué se utilizan las cookies?
    • Recordar información: Guardan tus preferencias (idioma, color de la web, zona horaria…). Así, la próxima vez que entres no tienes que configurarlo todo otra vez.
    • Mantener la sesión abierta: Si entras en Gmail o en una red social, la cookie recuerda que ya has iniciado sesión y no tienes que escribir la contraseña cada vez.
    • Personalizar anuncios y contenidos: Algunas cookies registran lo que visitas para mostrarte anuncios o recomendaciones relacionadas.
    • Idea Clave
      • Las cookies no son virus ni programas peligrosos.
      • Sirven para que la navegación sea más cómoda, pero también recogen datos sobre ti, por eso ahora las webs te preguntan si aceptas las cookies.

6. ¿Qué es un servidor WEB y para qué se utilizan?

  • Cuando visitas una página web, los datos por los que navegas han de estar almacenados en algún sitio para que puedas acceder.
  • En lugar de guardarlos en el ordenador del creador del sitio (esto supondría muchos problemas para el mismo), lo que se hace es utilizar servidores web en los que se alojan todos los datos.
  • Por tanto, un servidor web es un ordenador especialmente preparado para estar encendido las 24 horas del día los 365 días del año.
