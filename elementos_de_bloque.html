Elementos de Bloque

  • Los elementos de bloque se utilizan para construir la estructura principal de una página web, como los encabezados, párrafos y listas.
  • Estos comienzan en una nueva línea y ocupan todo el ancho disponible de la página.
  • Esto significa que cualquier contenido colocado dentro de un elemento de bloque crea un nuevo bloque independiente, separado del contenido anterior o posterior.

Ejemplos de elementos de bloque:

  • <h1> a <h6> → encabezados
  • <p> → párrafos
  • <ul> y <ol> → listas ordenadas y desordenadas
  • <div> → agrupación de contenido
