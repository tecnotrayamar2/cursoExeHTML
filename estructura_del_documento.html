Estructura del Documento

Estructura Básica de una Página WEB

En la estructura de una página web tenemos tres partes diferenciadas: cabecera, cuerpo y pie de página. O, por sus términos en inglés: header, body y footer.

Cabecera o Header

La cabecera es la parte superior de la página web. Aquí suele incluirse la información básica de la empresa o marca y es consistente en todo el sitio, es decir, se repite en cada página de la web por la que navegas.

Los elementos que se incluyen en la cabecera son:

  • Logo de la empresa
  • El menú de navegación
  • Un cuadro de búsqueda

 

 

 

 

 

Cuerpo o Body

Esta es la parte de la web que alberga el contenido principal de tu página. Esta parte sí es diferente en cada página de tu sitio. Es decir, no será el mismo contenido el que incluyes en tu página de contacto que en la página principal.

Para dividir u organizar la información en el body se suelen utilizar en la actualidad etiquetas semánticas.

 

 

 

 

 

 

 

Pie de Página o Footer

El pie de página o footer es la parte inferior de una página web. Al igual que la cabecera, se repite y es consistente en cada página. En el footer podemos encontrar los siguientes elementos:

  • Menú simplificado
  • Información de contacto
  • Botones de redes sociales
  • Logo de la empresa??
  • Enlace a la política de privacidad

 

 

 

 

 

 

 

Ejercicio 1: Mi Primera Página WEB

Duración:
30:00

El objetivo de este primer ejercicio es poner en práctica los contenidos vistos hasta el momento mediante la realización de nuestra primera página WEB. Los pasos a seguir son:

1. Crear el archivo HTML

Abrir Visual Studio Code.
Crear una carpeta nueva llamada MiPrimeraWeb.
Dentro, crear un archivo llamado index.html.

2. Generar la estructura básica

Dentro de index.html, <html> y pulsa Tabulador.
Aparecerá la etiqueta de cierre </html>.
Dentro de la etiqueta <html> introducimos las etiquetas <head> y <body>

3. Modificar el contenido

Cambia el título:
Dentro <head> del escribimos el título de nuestra página
<title> Mi primera página </title>
Dentro del <body>, escribe:
<h1> Hola, soy [tu nombre] </h1>
<p> Esta es mi primera página web hecha en clase. </p>

4. Abrir con Live Server

Haz clic derecho sobre el archivo index.html → Open with Live Server.
Se abrirá en el navegador.

5. Probar la actualización automática

Cambia el texto del párrafo y guarda con Ctrl + S.
Observa cómo el navegador se actualiza solo.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página WEB</title>
</head>
<body>
    <h1>Hola, soy Óscar</h1>
    <p>Esta es la primera página hecha en clase</p>
</body>
</html>

Importante

Recordamos

En los lenguajes naturales como el inglés, el español o el francés, la sintaxis consiste en seguir las reglas de gramática para colocar las palabras en el orden correcto y usar la puntuación adecuadamente, de modo que las oraciones tengan sentido.

De manera similar, la sintaxis en HTML se refiere a las reglas, directrices y convenciones que definen cómo debe estructurarse y escribirse el código HTML. Al seguir la sintaxis correcta, te aseguras de que tu código HTML sea entendido correctamente por los navegadores web y se muestre tal como deseas.

La sintaxis de HTML define la forma correcta de usar las etiquetas, atributos y valores (aprenderás qué son dentro de un momento) para crear páginas web que sean legibles y funcionales para los navegadores. Comprenderla es esencial para crear y mantener código HTML válido y eficaz.
