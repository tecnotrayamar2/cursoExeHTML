Atributos HTML

¿Qué es un atributo en HTML?

Un elemento HTML abarca todo desde la etiqueta de apertura hasta la etiqueta de cierre, incluyendo tanto el contenido como las etiquetas en sí.

La etiqueta de apertura marca el inicio de un elemento y puede incluir atributos, que son detalles adicionales que configuran o modifican el comportamiento del elemento.

Los atributos tienen un nombre y un valor, separados por un signo igual (=), y el valor se escribe entre comillas.

Como Usar los Atributos

Por ejemplo, la etiqueta <img> se utiliza para insertar una imagen en un documento HTML.

Tiene varios atributos que permiten especificar:

  • la fuente de la imagen,
  • su ancho y alto,
  • un texto descriptivo para propósitos de accesibilidad,
  • y otras propiedades.

En el caso de las imágenes los atributos más utilizados en son:

  • src, que indica la ruta del archivo de imagen, y
  • alt, que proporciona una descripción textual de la imagen, mostrada cuando la imagen no se puede cargar o visualizar.

Categorías de Atributos

Los atributos HTML pueden clasificarse en dos grandes categorías:

  • Atributos globales

Son aquellos que pueden aplicarse a cualquier elemento HTML.

  • Atributos específicos de elemento

Son atributos que solo se aplican a ciertos elementos HTML y no pueden utilizarse en todos.

Aportan funcionalidades o modifican el comportamiento de elementos concretos, por lo que son esenciales para su uso correcto.

Ejemplos:

  • href → específico del elemento <a> (enlaces)
  • src → específico de <img> y otros elementos multimedia
  • type → específico del elemento <input>
